# Programming Content Generator

A React + Vite web app for generating structured educational content for first-year programming students.

## Features
- Generate Lessons, Quizzes, Code examples, Assignments, Blog posts, and Emails
- Supports Python, Java, and JavaScript
- Difficulty levels: Beginner, Intermediate, Advanced
- Prompt Library with one-click prompt selection
- Custom prompt input
- Functional test cases panel

## Tech Stack
- React 18
- Vite 5
- Tailwind CSS 3

## User Preferences
- Keep component structure clean and split into separate files
